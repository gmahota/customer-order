# Customer Order Dashboard

A clean Next.js 15 migration of the original Nuxt/Vuetify repository. The app uses TypeScript, Tailwind CSS, and shadcn-style UI primitives to deliver a modern dark operations dashboard.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui-style components
- Mock customer and order data

## Routes

- `/dashboard`
- `/customers`
- `/orders`
- `/orders/new`

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```
